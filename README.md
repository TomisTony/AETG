# README

本项目为AETG算法的实现。支持n-pairwise。

# INSTALL

### python版本 

由于开发环境中的Python版本为python 3.10，因此建议使用3.10及以上版本运行

### 安装依赖包

```shell
pip install -r requirements.txt
```

# RUN

- 进入`code`文件夹

```shell
cd code
```

- 使用python启动脚本

```shell
python AETG.py
```

即可在`code`文件夹下自动生成包含京东与携程的2-pari以及3-pair测试数据的`.csv`文件。
